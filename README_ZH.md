# StylusFlow

> 致力于为 Mac 版 MarginNote 4 提供流畅的手写体验 | [English](README.md)

## 概述

**StylusFlow** 是一个 MarginNote 4 插件，让你可以为画布工具（笔、荧光笔、橡皮等）绑定自定义键盘快捷键，并提供 PDF 的HandMode与一些优化上的体验，带来更流畅的手写体验。

---


## 快速开始

### 安装

从 [GitHub Releases](https://github.com/yourusername/stylusflow/releases) 下载最新的 `.mnaddon` 文件，在 MarginNote 4 中导入。

### 从源码安装

```bash
pnpm install
pnpm dev
```

---

## 使用方法

1. 打开 StylusFlow 浮动面板
2. 切换到「快捷键」标签
3. 点击要绑定的工具
4. 按下你想要的组合键
5. 确认保存

在「调试」标签可以查看快捷键与工具检测状态，也可以管理 PDF 平移拦截、测试平移方向，并标定/切换脑图框选模式。发布版本默认不输出调试日志。

当遇到不可解决的问题时请在Issue区描述你的问题, 为了确保沟通效率,请清晰的描述你遇到的问题。
